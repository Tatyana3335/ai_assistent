## Описание проекта

Проект "ИИ-ассистент домашний помощник" представляет собой интеллектуального помощника, который помогает пользователям в решении повседневных задач. Ассистент способен вести осмысленный диалог, отвечать на вопросы на основе базы знаний, а также выполнять практические задачи, такие как запись на встречу и добавление событий в Google Календарь.

Этот проект разработан для улучшения качества жизни пользователей за счет автоматизации рутинных задач и предоставления быстрых и точных ответов на вопросы. Благодаря использованию современных технологий искусственного интеллекта, ассистент способен адаптироваться к потребностям пользователя и предоставлять персонализированные услуги.

---

## Функционал

### 1. **Осмысленное ведение диалога с помощью нейросети**
Ассистент способен поддерживать естественный и логичный диалог с пользователем. Он понимает контекст разговора, задает уточняющие вопросы при необходимости и предоставляет релевантные ответы. Это достигается за счет использования продвинутых языковых моделей.

### 2. **Ответы на вопросы по базе знаний**
Пользователь может задавать вопросы на различные темы, и ассистент будет отвечать на основе предварительно созданной базы знаний. База знаний включает информацию, необходимую для решения типовых задач пользователя, таких как советы по дому, рекомендации по планированию времени и другие полезные данные.

### 3. **Запись клиента на встречу и добавление встречи в Google Календарь**
Ассистент может помочь пользователю организовать его расписание. Для этого он:
- Уточняет детали встречи (дата, время, участники).
- Автоматически добавляет событие в Google Календарь.
- Отправляет напоминания о предстоящих встречах.

### 1. **Qwen**
Сервис Qwen был использован для:
- Создания системного промпта, который определяет поведение ассистента и его стиль общения.
- Формирования и структурирования базы знаний, чтобы обеспечить точность и релевантность ответов.

[Ссылка на Qwen](https://qwen.ai/)

### 2. **Савви (Suvvy)**
Сервис Suvvy был выбран для создания самого ИИ-ассистента. Он предоставляет удобные инструменты для интеграции различных функций, таких как диалоговый интерфейс и взаимодействие с внешними API (например, Google Календарь).

[Ссылка на Савви](https://suvvy.ai/)
